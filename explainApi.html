  • 文档说明

    • 背景
      • 为了方便本系统人员开发和其它系统调用本系统业务接口，编制了本接口文档。
    • 接口规范使用说明
      • 本文档为auPay平台为外部系统调用本系统提供的标准接口文档，详细说明和约束了各个接口的参数和返回值。阅读对象为需要调用本支付接口的开发人员。
    • 术语解释
      • 术语 英文 定义
        合作方 Api Key 调用API所使用的的密钥
        商户 Merchant 指委托收单的营业机构
        用户 User 指在商户系统内进行消费的企业或者个人
        商户号 MerchantId 指支付系统为使用外部接入接口的商户统一分配的唯一标识
        商户密钥 Api Secrect 指商户在和系统进行数据签名认证的秘钥，采用RSA加密算法

  • 接口说明

    • 支付接口
      • 1.概述
      • 功能描述：该接口用于调用支付接口进行支付，调用接口，会直接调转到金桃支付系统，进行支付，成功返回支付成功，失败返回错误信息。
      • 协议：HTTP
      • 请求类型：POST
      • 接收介质类型：application/json
      • 返回介质类型：application/json
      • 编码：UTF-8
      • 2.请求URL
      • a.接入URL：http://aupay.one/api/payment/order/payment
      • b.请求协议参数
      • 参数名 参数 参数类型 必填 说明
        商户号 merchantId VARCHAR true 商户号：商户在接入auPay时分配唯一商户号
        订单编号 merchantPaymentNo VARCHAR true 商户订单号，需要具有唯一性
        订单金额 paymentAmount DECIMAL true 订单金额（单位，元）请至少符合一种渠道限额
        商家同步通知地址 returnUrl VARCHAR true 支付成功后界面显示的地址，如果没有则使用商户申请地址
        商家异步通知地址 notifyUrl VARCHAR true 支付成功后，调取商家的接口，通知商家支付成功，优先使用商户后台设置的异步通知地址，如未设置则使用该参数。
      • c. 请求头
      • 参数名 参数 参数类型 必填 说明
        参数签名 AUPAY-SIGN VARCHAR true 根据请求参数签名

    • 提币接口
      • 1.概述
      • 功能描述：该接口用于调用支付接口进行支付，调用接口，会直接调转到金桃支付系统，进行支付，成功返回支付成功，失败返回错误信息
      • 协议：HTTP
      • 请求类型：POST
      • 接收介质类型：application/json
      • 返回介质类型：application/json
      • 编码：UTF-8
      • 2. 请求URL
      • a.接入URL：http://aupay.one/api/payment/order/withdraw
      • b.请求协议参数
      • 参数名 参数 参数类型 必填 说明
        商户号 merchantId VARCHAR true 商户号：商户在接入auPay时分配唯一商户号
        订单编号 merchantDrawNo VARCHAR true 商户订单号，需要具有唯一性
        订单金额 drawAmount DECIMAL true 订单金额，单位：元(与数币数量二选一)
        数币数量 currencyAmount DECIMAL true 数币数量，单位：USDT(与订单金额二选一)
        商家同步通知地址 returnUrl VARCHAR false 支付成功后界面显示的地址，如果没有则使用商户申请地址
        商家异步通知地址 notifyUrl VARCHAR false 支付成功后，调取商家的接口，通知商家支付成功，优先使用商户后台设置的异步通知地址，如未设置则使用该参数。
        提币渠道 drawChannelId VARCHAR false 提币渠道，转账币种
        提币地址 receiveAddress VARCHAR false 提币地址
      • c. 请求头
      • 参数名 参数 参数类型 必填 说明
        参数签名 AUPAY-SIGN VARCHAR true 根据请求参数签名

    • 交易查询
      • 1.商户充值交易查询 (因请求内容包含merchantId和apiKey 不建议直接在前端进行请求)
      • a.接入URL：http://aupay.one/api/payment/getPaymentInfo
      • b.接入方式：get方式
      • c.交易查询请求协议参数
      • 参数名 参数 参数类型 必填 说明
        订单编号 paymentOrderId or merchantPaymentNo VARCHAR true 商户订单号或系统订单号2选1,如果两个参数都传按照paymentOrderId获取
        商户号 merchantId VARCHAR true auPay平台商户唯一ID
        Api密钥 apiKey VARCHAR true API请求密钥
      • d.返回协议参数
      • 参数名 参数 参数类型 说明
        返回码 code int 返回码（200成功 201..等错误码）
        返回信息 message VARCHAR 返回信息（包含错误类型说明）
        订单信息 data jsonobj 见下方说明
      • e. data参数说明
      • 参数名 参数 参数类型 必填 说明
        订单编号 merchantPaymentNo VARCHAR true 商户在auPay平台唯一ID
        订单金额 amount DECIMAL true 订单金额，单位：元
        商户编号 merchantId VARCHAR true auPay平台商户唯一ID
        系统编号 paymentOrderId VARCHAR true auPay平台订单唯一ID
        订单状态 status int true 1到账0未到账
      • 2.商户提币交易查询 (因请求内容包含merchantId和apiKey 不建议直接在前端进行请求)
      • a.接入URL：http://aupay.one/api/payment/getDrawInfo
      • b.接入方式：get方式
      • c.交易查询请求协议参数
      • 参数名 参数 参数类型 必填 说明
        订单编号 drawOrderId or merchantDrawNo VARCHAR true 商户订单号或系统订单号2选1,如果两个参数都传按照paymentOrderId获取
        商户号 merchantId VARCHAR true 商户在auPay平台唯一ID
        Api密钥 apiKey VARCHAR true API请求密钥
      • d.返回协议参数
      • 参数名 参数 参数类型 说明
        返回码 code int 返回码（200成功 201..等错误码）
        返回信息 message VARCHAR 返回信息（包含错误类型说明）
        订单信息 data jsonobj 见下方说明
      • e. data参数说明
      • 参数名 参数 参数类型 必填 说明
        订单编号 merchantDrawNo VARCHAR true 商户在auPay平台唯一ID
        订单金额 amount DECIMAL true 订单金额，单位：元
        商户编号 merchantId VARCHAR true auPay平台商户唯一ID
        系统编号 drawOrderId VARCHAR true auPay平台订单唯一ID
        订单状态 status int true 1已到账 2已汇出 3处理中 0失败
